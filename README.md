# Tic Tac Toe

<p align="center"><img src="./thumbnail.png"></p>

Permainan Tic Tac Toe dengan algoritme minimax.

## Instalasi

```
git clone https://github.com/kangzoel/minmax-tictactoe.git
cd minmax-tictactoe
yarn
```

Development server: `yarn dev`
